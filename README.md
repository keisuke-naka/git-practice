# git-practice
repo for git practice

# git command
## クローン
```git clone クローン用URL```

## ブランチ切替
```git checkout ブランチ名```

## ブランチ作成
```git branch ブランチ名```

## 作業内容のステージング
```git add ステージングしたいファイル```

ディレクトリ内の全ての変更をステージングしたい場合

```git add .```

## 作業内容のコミット
```git commit -m "コメント"```

## コミットした内容をリモートリポジトリへプッシュ
```git push origin ブランチ名```

## コミット履歴の表示
```git log```
